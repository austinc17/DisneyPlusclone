
# 🎬 Disney+ Clone (In Progress)

This is a **React-based clone of Disney+**, designed to mimic the look and feel of a modern streaming service. The application dynamically fetches and displays trending movies using a movie API and features an interactive movie slider on the homepage.

> ⚠️ **Note:** This project is currently in development. Some features are still being built.

---

## 🚀 Features

- 🧭 **Interactive Trending Slider**  
  Displays trending movies that update each time the app is launched.

- 🔄 **Dynamic API Integration**  
  Fetches real-time data from a movie API to populate content.

---

## 🛠 Planned Features

- 🎞 **Genre-Based Sections**  
  Dedicated rows for genres like Action, Comedy, Drama, etc.

- 🔍 **Search and Filter**  
  A search bar with the ability to filter movies by genre, popularity, and more.

---

## 🧰 Tech Stack

- React.js  
- Tailwind CSS
- Movie API (e.g., TMDB API)
